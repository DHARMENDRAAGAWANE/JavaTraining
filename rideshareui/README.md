# Rideshareui
Car sharing is a good way to use up the full seating capacity of a car, which would otherwise remain unused if it were just the driver using the car. Delta Rideshare application allows any Delta Employee or contractor registered for carpool
 without worrying about leaking private information to public. A user can register as ride provider or car rider.
